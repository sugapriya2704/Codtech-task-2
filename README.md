# Codtech-task-2
Unified Data Integration &amp; Analytics Dashboard Project Overview

Synapse BI is a comprehensive Power BI project designed to integrate data from multiple sources and provide actionable business insights. By combining datasets from Excel and SQL Server, this dashboard enables organizations to analyze sales performance, customer behavior, and product trends in a single interactive view.

This project demonstrates data integration, transformation, and visualization skills, making it suitable for business intelligence reporting and decision-making processes.

Objectives

Integrate multiple data sources to create a unified dataset.

Provide insights into sales trends, customer segmentation, and regional performance.

Demonstrate Power BI modeling, DAX measures, and interactive reporting.

Deliver a professional dashboard suitable for stakeholders and management.

Data Sources Source Type Key Columns Description SalesData.xlsx Excel OrderID, CustomerName, Product, Quantity, SalesAmount Contains transactional sales data CustomerDB SQL Server Table CustomerID, CustomerName, Region, Contact Contains customer details and region info

Integration Key: CustomerName field is used to link datasets, enabling cross-analysis between sales and customer information.

**Key Features ** Data Integration & Transformation

Merge Excel and SQL Server data in Power BI.

Clean, transform, and standardize data using Power Query.

Handle missing or inconsistent values for accurate reporting.

Interactive Dashboards

Filter and slice data by region, product, and customer.

Drill-down capability to explore detailed sales trends.

Summary metrics and KPIs for quick business insights.

Business Insights

Sales by Region: Identify high-performing regions.

Top Customers: Rank customers based on revenue.

Sales Trends Over Time: Monthly/quarterly analysis.

Product Analysis: Track best-selling products.

Advanced Analytics (Optional)

Dynamic measures using DAX (Total Sales, Average Order Value, Growth %)

Conditional formatting to highlight key metrics

Visual storytelling with charts and KPI cards

Power BI Visualizations Used

Clustered Column Chart: Regional Sales comparison

Bar Chart: Top 5 Customers by revenue

Line Chart: Sales trend over time

Cards/KPI Metrics: Total Sales, Total Orders, Total Customers

Slicers & Filters: Product, Region, Time period
